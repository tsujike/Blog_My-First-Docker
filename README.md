# Blog_My-First-Docker

Dockerの基礎を学びながらブログを執筆するプロジェクトです。

## このプロジェクトについて

ノンプログラマー向けに、Dockerの基礎から実践までを学ぶためのブログ記事を執筆しています。

## 前提条件

- Windows環境
- Docker Desktopがインストール済み
- 基本的なコマンドライン操作の知識

## 学習の進め方

1. 記事を順番に読む（特に**Docker Compose完全ガイド**は重要）
2. `examples`フォルダのサンプルを実際に動かす
3. エラーが出たら、トラブルシューティングを参照
4. 自分なりにカスタマイズしてみる

## おすすめの学習パス

### 初心者向け（推奨順序）
1. **アプリが動く仕組み** ⭐（まずここから！）
2. **Docker用語解説**（必須）
3. Docker基礎入門 → Docker基本コマンド
4. WordPress環境構築（簡単版）
5. Docker Compose完全ガイド

### 実践重視
1. **アプリが動く仕組み** ⭐（理解を深めるために）
2. **Docker用語解説**（必須）
3. Docker Compose完全ガイド（最重要）
4. WordPress環境構築（完全版）
5. MySQL学習環境構築（完全版）
